## 🛒 DevClub Store

A DevClub Store é uma landing page moderna de e-commerce desenvolvida com foco em apresentar produtos de forma visual, organizada e atrativa. O projeto simula uma loja online, destacando itens, promoções e uma experiência de navegação simples e intuitiva.

## ✨ Sobre o Projeto

Este projeto foi criado com o objetivo de praticar e demonstrar habilidades em desenvolvimento web front-end, utilizando conceitos de layout, responsividade e design moderno.

A proposta é oferecer uma experiência semelhante a uma loja virtual real, com organização de produtos e elementos visuais que chamam a atenção do usuário.

## 🚀 Funcionalidades

- 🛍️ Exibição de produtos em destaque
- 🎯 Layout moderno e atrativo
- 📱 Design responsivo para diferentes dispositivos
- 🧭 Navegação simples e intuitiva
- 🎨 Uso de cores e elementos visuais bem definidos
- 🖱️ Interações visuais (hover e efeitos)

## 🛠 Tecnologias Utilizadas

- 💻 HTML5
- 🎨 CSS3
- ⚙️ JavaScript

O projeto foi desenvolvido com foco em responsividade, garantindo uma boa experiência tanto em:

- 📱 Smartphones
- 💻 Desktops
- 📲 Tablets
- 🎯 Objetivo

O principal objetivo do projeto é:

- Praticar desenvolvimento front-end
- Criar uma interface de loja virtual
- Melhorar habilidades em design e responsividade
- Simular um projeto real para portfólio
